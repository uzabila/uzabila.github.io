Как сменить домен в Google Search Console?

В Google Search Console есть встроенная опция, позволяющая в несколько кликов сменить домен (смена адреса сайта, перенаправление доменной собственности) на другой.

1. Завершите перенос веб-сайта со старого сайта на новый с помощью 301 редиректа

Обычно владельцы сайтов делают перенос сайта по следующим причинам:

  • смены адреса доменного имени сайта, при этом сам сайт (его содержимое) остается прежним;
  • перенаправление одного сайта на другой в результате слияния одного сайта с другим (обычно после этого часть контента со старого сайта добавляется на новый).

Существуют десятки нюансов, на которые следует обратить внимание при миграции сайта, но это не тема данного руководства. Для правильного завершения миграции используйте контрольный список из Search Console Help и Website Migration Checklist из Sitechecker.

2. Найдите в настройках сайта опцию смены адреса

Зайдите в Google Search Console и выберите проект, который вы хотите перенести на другое доменное имя. Затем нажмите «Настройки» (Settings) и «Изменить адрес» (Change of address).

3. Выберите собственность, которая будет новым доменом

Вы можете переместить доменную собственность только для тех доменов, к которым у вас есть доступ в Google Search Console. После выбора сайта нажмите кнопку «Проверить и обновить» (Validate & Update).

4. Отслеживайте, как проходит процесс миграции

Через некоторое время вы увидите уведомление о смене адреса внутри перенаправленного объекта (домена, доменной собственности).

Аналогичные уведомления вы увидите и в свойствах нового домена.

Вы также можете посетить опцию «Смена адреса» (Change of address) в «Настройках» (Settings) нового домена и найти информацию обо всех переезжающих сайтах на этот домен.
